# 🌟 Personal Portfolio Website

A modern, responsive, and animated personal portfolio website built using **HTML**, **CSS**, and **JavaScript**. This portfolio showcases my education, technical skills, projects, and contact information with a clean and professional design.

## 🚀 Features

- ✨ Responsive Design
- 🎨 Modern Glassmorphism UI
- ⌨️ Typing Animation
- 🌟 Smooth Scroll Animations (AOS)
- 💻 Skills Section
- 📚 Education Timeline
- 🚀 Featured Projects
- 📄 Resume Download Button
- 📞 Contact Section
- 📱 Mobile Friendly

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
- AOS (Animate On Scroll)
- Typed.js
